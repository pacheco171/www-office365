Como configurar o Azure AD
1
Acesse App registrations → New registration
Nome: "LIVE M365 Dashboard", Tipo: "Accounts in this organizational directory only"
2
Na pagina do app, copie o Application (client) ID e Directory (tenant) ID da aba Overview
3
Acesse Certificates & secrets → New client secret → copie o Value
4
Acesse API permissions → Add a permission → Microsoft Graph → Application permissions:
User.Read.All Directory.Read.All Reports.Read.All
5
Clique em Grant admin consent para aprovar as permissoes
6
Cole as credenciais nos campos ao lado e clique em Testar Conexao
